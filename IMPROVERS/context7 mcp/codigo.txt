Context7 MCP — provides up-to-date, version-specific documentation for libraries directly into your prompts.
Usage: add "use context7" in your prompt to fetch live docs for any library.

Claude Code:
claude mcp add context7 -- npx -y @upstash/context7-mcp

Claude (global scope):
claude mcp add --scope user context7 -- npx -y @upstash/context7-mcp

Claude (project scope):
claude mcp add --scope project context7 -- npx -y @upstash/context7-mcp

JSON config (claude_desktop_config.json / .mcp.json):
{
  "mcpServers": {
    "context7": {
      "command": "npx",
      "args": ["-y", "@upstash/context7-mcp"]
    }
  }
}

With uvx (alternative):
{
  "mcpServers": {
    "context7": {
      "command": "uvx",
      "args": ["context7-mcp"]
    }
  }
}
